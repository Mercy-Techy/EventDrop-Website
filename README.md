# EventDrop Website

A website for creating and managing event-based photo galleries. Event owners can create events, generate unique upload links, and allow guests to upload photos without authentication. Supports real-time updates using Socket.IO, photo moderation, likes/comments, and optional premium features.

## Features

Create and manage events

Generate unique public upload links

Guests can upload photos without login

Real-time photo updates using Socket.IO

Event owners can approve photos before they go live

Users can like and comment on photos

Event expiration for non-premium accounts

Automatic watermarking

Photographer high-quality upload support

## Tech Stack

React.js

TailwindCSS

TypeScript
